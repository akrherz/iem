The Ani(mation)S applet

This simple form will just allow the users to stop and start the animation, zoom and change the animations speed. The zooming will also allow for roaming (mouse drag while zoomed).

The complete applet tag is:

<APPLET code="AniS.class" width=640 height=540> <PARAM name="controls" value="startstop,zoom,speed"> <PARAM name="filenames" value="fsd_rad_0,fsd_rad_1,fsd_rad_2"> </APPLET>
